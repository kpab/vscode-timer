# Time Tracker

A modern time tracking extension for Visual Studio Code that helps you monitor the time spent on different files in your projects.

**Time Tracker** は、VSCode 上でファイルごとの作業時間を自動的に記録し、美しいビジュアライゼーションやフローティングタイマーなど、多彩な機能を持つ時間管理ツールです。

## 🌟 Features | 機能

### 1. Automatic Time Tracking | 自動時間計測
- Automatically tracks time when you switch between files
- Persists data across VSCode sessions
- File-specific time tracking

- ファイル切り替え時に自動的に時間を計測
- VSCodeセッション間でデータを保持
- ファイルごとの個別計測

### 2. Status Bar Display | ステータスバー表示
- Shows total time and current file time at a glance
- Click to open the Time Tracker panel

- 合計時間と現在のファイルの時間を一目で確認
- クリックでTime Trackerパネルを開く

### 3. Beautiful Time Cards | 美しいタイムカード
- Generate beautiful visual time cards with gradients and animations
- Export as HTML for sharing or printing

- グラデーションとアニメーションの美しいタイムカードを生成
- 共有や印刷用のHTML形式でエクスポート

### 4. Floating Timer | フローティングタイマー
- Always-visible floating timer panel
- Real-time updates every second
- Show/hide and pause/resume controls

- 常に表示されるフローティングタイマーパネル
- 1秒ごとのリアルタイム更新
- 表示/非表示、一時停止/再開コントロール

### 5. File Exclusion | ファイル除外機能
- Exclude specific files from time tracking
- Status bar indicator shows excluded files
- Manage excluded files easily

- 特定のファイルを時間計測から除外
- ステータスバーで除外ファイルを表示
- 除外ファイルを簡単に管理

### 6. Reset Functionality | リセット機能
- Reset all time data with one click
- Confirmation dialog to prevent accidental resets

- ワンクリックですべての時間データをリセット
- 誤操作防止の確認ダイアログ

## 🚀 Quick Start | クイックスタート

1. Install the extension | 拡張機能をインストール
2. Open any file in VSCode | VSCode でファイルを開く
3. Time tracking starts automatically | 自動的に時間計測が開始

## 📖 Usage Guide | 使い方ガイド

### Viewing Time Data | 時間データの確認
- **Status Bar**: Click the timer to open Time Tracker panel
- **Explorer Panel**: View time per file in the sidebar

- **ステータスバー**: タイマーをクリックしてTime Trackerパネルを開く
- **エクスプローラパネル**: サイドバーでファイルごとの時間を表示

### Generating Time Cards | タイムカードの生成
1. Click the preview icon in Time Tracker panel
2. View the beautifully designed time card
3. HTML is saved for sharing

1. Time Trackerパネルのプレビューアイコンをクリック
2. 美しくデザインされたタイムカードを確認
3. 共有用のHTMLが保存される

### Using Floating Timer | フローティングタイマーの使用
1. Click the floating timer icon in Time Tracker panel
2. Move the panel anywhere on screen
3. Use controls to pause/hide as needed

1. Time Trackerパネルのフローティングタイマーアイコンをクリック
2. パネルを画面上の任意の場所に移動
3. 必要に応じて一時停止や非表示機能を使用

### Managing Excluded Files | 除外ファイルの管理
1. Click the eye icon in status bar
2. Choose to exclude/include current file
3. Manage all excluded files from quick menu

1. ステータスバーの目アイコンをクリック
2. 現在のファイルを除外/含めるかを選択
3. クイックメニューからすべての除外ファイルを管理

### Resetting Data | データのリセット
1. Click the reset icon in Time Tracker panel
2. Confirm action in dialog
3. All time data is cleared

1. Time Trackerパネルのリセットアイコンをクリック
2. ダイアログで操作を確認
3. すべての時間データがクリアされる
